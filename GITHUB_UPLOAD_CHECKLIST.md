# GitHub Upload Checklist

Repository suggestion: `apollo-11-experience`
Visibility suggestion: **Public**, because GitHub Pages availability for private repositories depends on the account plan.

After the files are uploaded to the `main` branch:

1. Open **Settings → Pages**.
2. Set **Source** to **GitHub Actions**.
3. Open **Actions → Deploy Apollo 11 Experience**.
4. Run the workflow manually if the initial push did not start it.
5. Wait for both `build` and `deploy` to show green checks.
6. Open the URL displayed by the deployment job.
7. On iPhone Safari, choose **Share → Add to Home Screen**.

No secrets, API keys, domain records, or paid services are required.
